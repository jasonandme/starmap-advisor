# Security Policy

## Reporting

如果你发现安全问题，请优先通过私下渠道联系维护者，不要在公开 Issue 中贴出可利用细节、真实凭证、个人持仓截图或账户信息。

## Sensitive Data

请不要提交以下内容：

- LLM、数据源、云服务或数据库的真实 API Key
- `.env`、数据库文件、缓存、日志和上传文件
- 个人身份信息、账户截图、交易记录或完整持仓截图
- 能够定位个人账户、手机号、邮箱或资金规模的样例数据

## Runtime Notes

项目默认用于本地研究和学习。若部署到公网环境，请至少补充：

- 强认证和访问控制
- HTTPS 与反向代理安全配置
- 上传文件大小、类型和内容校验
- 日志脱敏与保留周期
- 数据库备份和最小权限账号
- LLM 调用限流和成本保护
