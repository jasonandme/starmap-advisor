# Roadmap

这个路线图描述项目可以自然演进的方向，优先级会根据数据源稳定性、使用反馈和维护成本调整。

## 已具备

- 基金搜索、排名、详情和多基金对比
- 组合偏好、持仓/自选管理和策略提示
- 板块行情、资金流向和相关新闻观察
- 基于 LLM 的自然语言问答与技能调用
- Docker Compose 与本地开发脚本

## 近期计划

- 增加核心 API 的单元测试和数据适配器回归样例
- 优化数据源异常时的 warning、fallback 和重试策略
- 为前端补充空状态、加载态和错误态的一致体验
- 增加项目截图和典型工作流文档

## 中期计划

- 支持更多基金风险指标，例如最大回撤、夏普比率和波动率分解
- 引入 PostgreSQL 部署样例和数据库迁移工具
- 增加异步任务队列，用于批量刷新和慢查询
- 支持可配置的数据源优先级和数据新鲜度展示

## 长期方向

- 构建更完整的投研知识库和来源引用机制
- 支持策略回测和更严格的时间切分验证
- 提供插件化指标扩展接口
- 强化权限、审计和隐私保护能力
